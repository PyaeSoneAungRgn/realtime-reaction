# Realtime Reaction

POC project using [Laravel Reverb](https://reverb.laravel.com/) and [laravel-reactions](https://github.com/qirolab/laravel-reactions).

```bash
cp .env.example .env
php artisan migrate --seed
php artisan reverb:start
php artisan queue:listen
```
